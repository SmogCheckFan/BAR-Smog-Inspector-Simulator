# BAR Smog Check Inspector – Exam Simulator

Free, fully-functional exam simulator for California BAR Smog Check Inspector certification.

## About

Built by a CalArc-certified BAR Inspector (Level 1 & 2) using DeepSeek AI.
**320 high-quality questions** covering all exam topics.

Validated by independent AI review and expert assessment.

## Features

✅ Timed exam (150 minutes)
✅ 110 random questions per attempt (90 scored, 20 unscored)
✅ Instant feedback on every answer
✅ Flag questions for review
✅ Topic performance breakdown
✅ Auto-save progress to browser
✅ Mobile responsive
✅ No installation, no login, no ads

## How to Use

1. Download or open this repository
2. Open `index.html` in any modern web browser
3. Click "Start Exam"
4. Take your practice test
5. Review your performance
